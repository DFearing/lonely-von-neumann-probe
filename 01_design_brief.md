# Bobiverse Idle Game - UI Design Brief

## Game Overview
A space exploration idle game where players build and customize autonomous probes to spread across star systems. Players manage resources, research technologies, and expand their empire through independent systems that operate in parallel.

## Core Loop
1. **Resource Generation** - Probes mine materials, generate energy via reactors
2. **Structure Building** - Construct miners, reactors, 3D printers using materials
3. **Research** - Spend computing power to unlock tech that improves efficiency
4. **Probe Expansion** - Build and send new probes to colonize new star systems
5. **System Management** - Manage multiple independent systems simultaneously

## Key Design Constraints
- **Real-time idle progression** - Resources generate continuously, player can close and return
- **Independent systems** - Each star system operates separately with its own buildings, tech, and resources
- **Modular probes** - Players customize probes by combining CPU + Propulsion + Reactor components
- **Parallel construction** - Multiple 3D printers work on separate projects (until networking tech unlocks pooling)
- **Light-speed tech sharing** - Early game: tech propagates at light speed between systems; late game: instant with research

## Three Resource Types
- **Materials** - Mined from asteroids; used to build structures and probes
- **Energy** - Generated by reactors; required for construction and structure operation
- **Computing Power** - Generated by probes; determines research speed

## Primary User Actions
1. Build structures (miners, reactors, 3D printers)
2. Customize and launch new probes to other systems
3. Research technologies across 6 branching tech trees
4. View and manage multiple star systems
5. Monitor resource production and consumption rates
6. Queue construction projects

## UI Complexity Levels
- **Early Game** - Single system, simple UI, handful of options
- **Mid Game** - 2-3 systems, tech branching becomes visible, more customization
- **Late Game** - 5-10+ systems, complex tech trees, advanced probe configurations

## Aesthetic Direction
**Sci-fi Command Center** - Sleek, retro-futuristic aesthetic. Deep space theme with dark backgrounds, neon/cyan accents. Feel like you're managing a space fleet from mission control. Clean, readable fonts and data-forward layout.

## Key UI Metrics to Display
- Current resources (Materials, Energy, Computing Power) with production/consumption rates
- Active construction projects and queue
- Research progress and available techs
- System list with status/richness info
- Current probe composition and available components
- Time to completion for major projects
